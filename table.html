HTML Table

O que é uma Tabela em HTML?

As tabelas em HTML são usadas para organizar dados em linhas e colunas. Elas são representadas pela tag <table> e seus elementos relacionados, como <tr> (linhas), <th> (cabeçalhos de coluna) e <td> (dados de célula).

Exemplo de Uso de Tabela em HTML

Aqui está um exemplo simples de como uma tabela pode ser criada em uma página HTML:

          
            <table>
                <tr>
                    <th>Nome</th>
                    <th>Idade</th>
                </tr>
                <tr>
                    <td>João</td>
                    <td>25</td>
                </tr>
                <tr>
                    <td>Maria</td>
                    <td>30</td>
                </tr>
            </table>
