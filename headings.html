HTML Headings

O que são Headings em HTML?

Os headings, ou cabeçalhos, em HTML são elementos que representam os títulos ou subtítulos de uma página web. Eles são usados para estruturar o conteúdo e indicar a hierarquia das seções.

Exemplo de uso de Headings em HTML

Aqui está um exemplo simples de como os headings podem ser usados em uma página HTML:

          
            <h1>Título Principal</h1>
            <h2>Subtítulo</h2>
            <h3>Subtítulo Menor</h3>
            <h4>Subtítulo Menor ainda</h4>
            <h5>Subtítulo Menor ainda</h5>
            <h6>Subtítulo Menor ainda</h6>
