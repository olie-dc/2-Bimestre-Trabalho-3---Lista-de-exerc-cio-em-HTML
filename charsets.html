HTML Charsets

O que são Charsets em HTML?

Charsets, ou conjuntos de caracteres, em HTML são especificações que definem como os caracteres são codificados em uma página da web. Eles garantem que os navegadores exibam o texto corretamente, especialmente em páginas com diferentes idiomas e conjuntos de caracteres.

Exemplo de Uso de Charsets em HTML

Aqui está um exemplo simples de como especificar o charset em uma página HTML:

          
            <!DOCTYPE html>
            <html lang="pt-BR">
            <head>
                <meta charset="UTF-8">
                <title>Minha Página</title>
            </head>
            <body>
                <p>Este é um exemplo de página HTML com charset UTF-8.</p>
            </body>
            </html>
