HTML URL Encode

O que é Codificação de URL em HTML?

A codificação de URL em HTML é o processo de substituir caracteres especiais por códigos de escape em uma URL. Isso é útil quando você precisa incluir caracteres que não são permitidos em uma URL, como espaços, acentos ou símbolos.

Exemplo de Codificação de URL em HTML

Aqui está um exemplo simples de como codificar uma URL em HTML:

          
            <a href="https://www.example.com/search?q=lorem ipsum">Pesquisar</a>

Neste exemplo, a URL contém um espaço entre "lorem" e "ipsum", o que não é permitido. Para corrigir isso, podemos codificar a URL:

          
            <a href="https://www.example.com/search?q=lorem%20ipsum">Pesquisar</a>
