HTML Iframes

O que são Iframes em HTML?

Os iframes em HTML são usados para incorporar uma página web dentro de outra página. Eles permitem que você exiba conteúdo de outra fonte ou site dentro de uma área específica de sua página.

Exemplo de Uso de Iframes em HTML

Aqui está um exemplo simples de como um iframe pode ser usado em uma página HTML:

          
            <iframe src="https://www.example.com" width="400" height="300" frameborder="0">
                Seu navegador não suporta iframes.
            </iframe>
