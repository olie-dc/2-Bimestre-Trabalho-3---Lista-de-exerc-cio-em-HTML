HTML Entities

O que são Entidades HTML?

Entidades HTML são códigos especiais usados para representar caracteres reservados ou caracteres que não podem ser facilmente digitados no teclado. Elas são úteis para exibir símbolos, caracteres especiais e emojis em uma página web.

Exemplo de Uso de Entidades HTML

Aqui está um exemplo simples de como as entidades HTML podem ser usadas em uma página:

          
            <p>Para exibir o símbolo de copyright, você pode usar &copy; ou &#169;.</p>
            <p>Para exibir uma seta para a direita, você pode usar &rarr; ou &#8594;.</p>
            <p>Para exibir um emoji sorridente, você pode usar 😀.</p>

HTML implementado na própria página

Aqui está o código HTML acima implementado na própria página:

Para exibir o símbolo de copyright, você pode usar © ou ©.

Para exibir uma seta para a direita, você pode usar → ou →.

Para exibir um emoji sorridente, você pode usar 😀.
