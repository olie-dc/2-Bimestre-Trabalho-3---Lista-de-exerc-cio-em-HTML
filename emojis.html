HTML Emojis

O que são Emojis em HTML?

Emojis são pequenas representações gráficas de expressões faciais, objetos, símbolos, animais e muito mais. Em HTML, os emojis podem ser inseridos diretamente usando seus códigos Unicode ou através de entidades HTML.

Exemplo de Uso de Emojis em HTML

Aqui está um exemplo simples de como emojis podem ser usados em uma página HTML:

          
            <p>Você pode inserir emojis diretamente usando códigos Unicode, por exemplo: 😀.</p>
            <p>Também é possível usar entidades HTML para inserir emojis, como: &#128512; ou &#x1F600;.</p>
