HTML Drag/Drop

O que é o HTML Drag/Drop?

O recurso de arrastar e soltar em HTML permite que os usuários arrastem elementos HTML e os soltem em locais específicos na página. Isso pode ser útil para interfaces de usuário interativas, como a reordenação de itens em uma lista, o envio de arquivos por arrastar e soltar e outras interações dinâmicas.

Exemplo de Uso de Arrastar e Soltar em HTML

Aqui está um exemplo simples de uso da funcionalidade de arrastar e soltar em HTML:

          
            <div id="drag" draggable="true" ondragstart="event.dataTransfer.setData('text/plain', null)">
                Arraste-me!
            </div>

            <div id="drop" ondrop="drop(event)" ondragover="allowDrop(event)">
                Solte aqui!
            </div>

            <script>
                function allowDrop(event) {
                    event.preventDefault();
                }

                function drop(event) {
                    event.preventDefault();
                    var data = event.dataTransfer.getData('text/plain');
                    event.target.appendChild(document.getElementById(data));
                }
            </script>
