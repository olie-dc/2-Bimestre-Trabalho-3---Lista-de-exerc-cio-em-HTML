O que é HTML?

HTML (Hypertext Markup Language) é a linguagem padrão utilizada para criar páginas web. Ele define a estrutura e o conteúdo de uma página web usando uma série de elementos e tags.

Exemplo de código HTML básico

Aqui está um exemplo simples de um documento HTML básico:

        
            <!DOCTYPE html>
            <html>
            <head>
            <title>Título da Página</title>
            </head>
            <body>
            <h1>Meu Primeiro Título</h1>
            <p>Meu primeiro parágrafo.</p>
            </body>
            </html>
