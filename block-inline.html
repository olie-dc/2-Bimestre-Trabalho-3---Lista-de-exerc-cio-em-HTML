HTML Block & Inline

O que são Elementos de Bloco e Elementos Inline em HTML?

Em HTML, existem dois tipos principais de elementos: elementos de bloco e elementos inline.

Elementos de Bloco:

Os elementos de bloco são usados para criar estruturas de layout e geralmente ocupam toda a largura disponível na página. Eles começam em uma nova linha e empurram outros elementos para baixo. Alguns exemplos comuns de elementos de bloco incluem <div>, <p>, <h1>-<h6>, <ul>, <ol>, <li>, <table>, entre outros.

Elementos Inline:

Os elementos inline são usados para marcar partes de um texto e geralmente aparecem na mesma linha que o conteúdo ao seu redor. Eles não começam em uma nova linha e apenas ocupam o espaço necessário para seu conteúdo. Alguns exemplos comuns de elementos inline incluem <span>, <a>, <strong>, <em>, <img>, <input>, <br>, entre outros.

Exemplo de Elementos de Bloco e Inline em HTML

Aqui está um exemplo simples de como os elementos de bloco e inline podem ser usados em uma página HTML:

          
            <div style="background-color: #ccc;">
                Este é um elemento de bloco.
                <a href="#">Este é um link inline.</a>
                <p>Este é outro elemento de bloco dentro do div.</p>
            </div>
