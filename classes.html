HTML Classes

O que são Classes em HTML?

As classes em HTML são atributos que podem ser atribuídos a elementos HTML para estilizá-los ou selecioná-los com CSS ou JavaScript. Uma classe é definida usando o atributo `class` e pode ser reutilizada em vários elementos.

Exemplo de Uso de Classes em HTML

Aqui está um exemplo simples de como as classes podem ser usadas em uma página HTML:

          
            <style>
                .destaque {
                    color: red;
                    font-weight: bold;
                }
            </style>

            <p class="destaque">Este parágrafo está em destaque.</p>
            <p>Este parágrafo não está em destaque.</p>
