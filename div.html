HTML Div

O que é uma Div em HTML?

A tag <div> em HTML é usada para criar uma divisão ou seção genérica em uma página web. Ela é frequentemente usada para agrupar e estilizar elementos HTML com CSS, e não tem significado semântico específico.

Exemplo de Uso de Div em HTML

Aqui está um exemplo simples de como a tag <div> pode ser usada em uma página HTML:

          
            <div id="container">
                <h2>Título</h2>
                <p>Este é um parágrafo dentro da div.</p>
            </div>
