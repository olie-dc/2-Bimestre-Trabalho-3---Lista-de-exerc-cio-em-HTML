HTML vs XHTML

O que é HTML?

HTML (Hypertext Markup Language) é uma linguagem de marcação utilizada para criar e estruturar conteúdo em páginas da web. Ela é mais flexível em termos de sintaxe e não requer um XML bem formado.

O que é XHTML?

XHTML (Extensible Hypertext Markup Language) é uma versão mais rigorosa do HTML que segue as regras da sintaxe XML. Ele requer que todas as tags sejam fechadas corretamente e que o documento seja bem formado.

Exemplo de HTML

          
            <!DOCTYPE html>
            <html>
            <head>
                <title>Exemplo de HTML</title>
            </head>
            <body>
                <h1>Este é um exemplo de HTML.</h1>
                <p>HTML é mais flexível em relação à sintaxe.</p>
            </body>
            </html>

Exemplo de XHTML

          
            <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
            "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
            <html xmlns="http://www.w3.org/1999/xhtml">
            <head>
                <title>Exemplo de XHTML</title>
            </head>
            <body>
                <h1>Este é um exemplo de XHTML.</h1>
                <p>XHTML exige uma sintaxe mais rigorosa e bem formada.</p>
            </body>
            </html>

Conclusão

HTML e XHTML têm suas diferenças na sintaxe e nas regras de formatação. HTML é mais flexível, enquanto XHTML é mais rigoroso e segue as regras do XML.
