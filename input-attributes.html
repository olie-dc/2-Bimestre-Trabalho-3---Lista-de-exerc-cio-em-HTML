HTML Input Attributes

Exemplo de Formulário em HTML com Atributos de Entrada

Aqui está um exemplo de formulário em HTML com diferentes atributos para os elementos de entrada:

          
            <form action="/submit" method="post">
                <label for="text">Campo de Texto:</label>
                <input type="text" id="text" name="text" placeholder="Digite seu nome" maxlength="50" required autofocus>

                <label for="email">Email:</label>
                <input type="email" id="email" name="email" placeholder="Digite seu email" required>

                <label for="password">Senha:</label>
                <input type="password" id="password" name="password" minlength="6" required>

                <label for="date">Data:</label>
                <input type="date" id="date" name="date" min="2022-01-01" max="2025-12-31" required>

                <label for="checkbox">Checkbox:</label>
                <input type="checkbox" id="checkbox" name="checkbox" value="checked" checked>

                <label for="radio">Radio:</label>
                <input type="radio" id="radio1" name="radio" value="option1" checked>
                <label for="radio1">Opção 1</label>
                <input type="radio" id="radio2" name="radio" value="option2">
                <label for="radio2">Opção 2</label>

                <label for="file">Upload de Arquivo:</label>
                <input type="file" id="file" name="file" accept=".jpg, .jpeg, .png" multiple>

                <button type="submit">Enviar</button>
            </form>
