HTML JavaScript

O que é JavaScript em HTML?

O JavaScript em HTML é uma linguagem de programação de script frequentemente usada para criar interatividade em páginas web. Ele pode ser incorporado em documentos HTML para manipular o conteúdo da página, responder a eventos do usuário e muito mais.

Exemplo de Uso de JavaScript em HTML

Aqui está um exemplo simples de como JavaScript pode ser usado em uma página HTML:

          
            <button onclick="myFunction()">Clique aqui</button>

            <script>
                function myFunction() {
                    alert("Olá! Este é um exemplo de JavaScript em HTML.");
                }
            </script>
