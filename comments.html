HTML Comments

O que são Comentários em HTML?

Os comentários em HTML são usados para adicionar notas explicativas no código HTML que não são exibidas no navegador. Eles são úteis para documentar o código, fazer anotações ou desativar temporariamente partes do código.

Exemplo de Comentários em HTML

Aqui está um exemplo simples de como os comentários podem ser adicionados em uma página HTML:

          
            <!-- Este é um comentário HTML -->
            <p>Este é um parágrafo de texto.</p>
