HTML SVG

O que é a tag SVG?

A tag SVG (Scalable Vector Graphics) é usada em HTML para desenhar gráficos vetoriais escaláveis. Ela permite criar gráficos complexos, como gráficos de barras, gráficos de pizza e até mesmo gráficos animados, usando marcação XML. Os gráficos criados com SVG mantêm sua qualidade e clareza independentemente do tamanho em que são exibidos, tornando-os ideais para aplicações web responsivas.

Exemplo de Uso da Tag SVG

Aqui está um exemplo simples de uso da tag SVG em HTML:

          
            <svg width="200" height="200">
                <circle cx="100" cy="100" r="80" fill="red" />
                <rect x="20" y="20" width="160" height="160" stroke="black" stroke-width="3" fill="none" />
            </svg>
